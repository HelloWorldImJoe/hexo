		*  @return 如果是我们想要的 -> return Yes; 否则 -> return No;
 		*/

2. `- (void)didSelectPost`

		///点post(发布)之后调用的方法
		- (void)didSelectPost {
		
    		// This is called after the user selects Post. Do the upload of contentText and/or NSExtensionContext attachments.

		}

### 分享网页内容
